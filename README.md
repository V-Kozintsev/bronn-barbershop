# БРОНН Барбершоп

Адаптивный многостраничный сайт барбершопа БРОНН: первый экран, услуги и прайс, карточки мастеров, карусель команды, выбор услуги или мастера, форма онлайн-записи, карта, контакты, политика конфиденциальности и cookie-уведомление.

Посмотреть сайт: https://v-kozintsev.github.io/bronn-barbershop/

## Что использовалось

- Vue 3 и CSS
- Vite для локального запуска и сборки
- Vue Router для страниц и маршрутизации
- lucide-vue-next для иконок интерфейса
- GitHub Pages для публикации

## Локальный запуск

```bash
pnpm install
pnpm dev
```

## Сборка

```bash
pnpm build
```
