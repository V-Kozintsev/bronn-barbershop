# Бронн Барбершоп

Многостраничный Vue/Vite-сайт вымышленного барбершопа в Санкт-Петербурге.

## Стек

- Vue 3
- Vue Router
- Vite
- lucide-vue-next
- GitHub Pages

## Команды

```bash
pnpm install
pnpm dev
pnpm build
pnpm preview
```

## Страницы

- Главная
- Услуги и прайс
- Мастера
- Запись и контакты

## Публикация

Workflow `.github/workflows/deploy.yml` собирает проект и публикует папку `dist` через GitHub Pages.
