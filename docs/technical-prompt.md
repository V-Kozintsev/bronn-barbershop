# Технический prompt проекта

Создать многостраничный Vue/Vite-сайт вымышленного барбершопа «Бронн» для российской аудитории, Санкт-Петербург. Сайт должен восприниматься как настоящий бизнес, без пометок о демонстрационности.

Требования:
- Не одностраничный лендинг: главная, услуги/прайс, мастера, запись/контакты.
- GitHub Pages-ready: относительный `base`, статическая сборка, без backend.
- Визуальный современный дизайн с премиальным, но не перегруженным настроением.
- Адаптивы от маленького телефона до ультраширокого desktop.
- Собственные отдельные bitmap-изображения для hero, услуг, мастера и атмосферы.
- SVG не рисовать вручную; для иконок использовать библиотеку.
- Навигация, кнопки, карточки, телефоны, мессенджеры, карта и формы должны быть кликабельными.
- Форма записи работает на фронтенде: валидация, состояние отправки, успешная отправка.
- Красивое раскрывающееся burger-меню для телефона и планшета.
- Карта: Санкт-Петербург, Яндекс.Карта через iframe.
- Footer с юридическим блоком и нижней полосой «Сайт разработан от Pusk Web» со ссылкой-заглушкой.
- Без резких hover-анимаций и тяжелых эффектов.
